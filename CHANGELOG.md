Please download and install the `EDM_TOOLS_vx.x.x.zip`, **Not** the `Source code (zip)` for the plugin to install correctly

---

This build features the Rig Clickable module, to remove the repetitive steps of making clickables, it won't animate your switches for you, but it will preconfigure all the empties, names, actions etc into the DCS format.

---

## Changelog:
* **Added:** Rig Clickables Module
    * Set Clickable Arg
    * Set Action Name
    * Set Clickable Name (or leave default for DCS Standard Convention)
    * Auto Generates action (correctly named)
    * Auto Generates clickable box, to roughly correct size
    * Auto handles parenting.
    * Cleans up created empties into seperate collection.
    * Interface with ED .edm exporter to set the clickables box to the correct object type (connector)
